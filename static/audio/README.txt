# 背景音乐下载指南

为了使视频生成功能中的背景音乐选项正常工作，请按照以下步骤操作：

## 下载步骤

1. 从以下免费音乐网站下载一个您喜欢的MP3背景音乐文件：
   - [Pixabay音乐](https://pixabay.com/music/)
   - [Mixkit](https://mixkit.co/free-stock-music/)
   - [Fesliyan Studios](https://www.fesliyanstudios.com/royalty-free-music/downloads-c/peaceful-and-relaxing-music/22)

2. 将下载的音乐文件重命名为 `background.mp3`

3. 将重命名后的文件放在本应用的 `static/audio` 目录下

## 注意事项

- 请确保您下载的音乐文件是免费可商用的，或者您拥有使用权
- 建议选择舒缓、轻松的背景音乐，以配合漫画视频的氛围
- 音乐文件大小建议不超过10MB，以保证应用性能
- 支持的格式为MP3或WAV

如果您不想使用背景音乐，可以在视频生成界面取消勾选"添加背景音乐"选项。
